# Card images

Drop your images here with these exact filenames and the cards pick them up
automatically (no code change needed). Until a file exists, the card shows a
soft tinted glow orb in its place, so nothing looks broken.

| Card | Filename | Recommended |
|------|----------|-------------|
| Inner Cosmos (feature) | `inner-cosmos.png` | wide, ~1200×825 (16:11) |
| Neuroscience for Kids | `kids.png` | square, ~600×600 |
| The Cosmos, on a Wall | `wall.png` | square, ~600×600 |
| Scales of the Brain | `scales.png` | square, ~600×600 |
| Sources & Citations | `citations.png` | square, ~600×600 |

Notes
- PNG or JPG both work; if you use `.jpg`, update the `src` in `index.html`.
- Images are shown with `object-fit: cover`, so they fill the slot and crop to
  fit — keep the subject roughly centered.
- They read best on the dark background if they're vivid / high-contrast.
